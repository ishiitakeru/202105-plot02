# 宇宙難民の話

## 墓参り
* 主人公が父親の墓を参る
* 父の元同僚に出会う
* エージェントとしての適性検査のために血液サンプルを提供する

## 宇宙船不時着
* 主人公が宇宙船の不時着に遭遇する
* エイリアンを保護する
* エイリアンはよその星の侵略に失敗した悪の組織の科学担当であることが判る

## 入管ヒーローとの戦い
* 父の元同僚がやってくる
* 適性検査の結果、高い数値が出たのでエージェントとしてスカウトしたい
* 父の元同僚が変身する
* 適性者に許された範囲での情報提供
  * 地球にはすでにエイリアンが住んでいる
  * 現在ではエイリアンはすべて排除の方針なので最初の仕事はそのエイリアンの駆除
* 主人公はスカウトを断り、エイリアンを守るべく立ち向かう
* エイリアンは侵略用の技術である現地生物兵器化のガジェットを主人公に提供。主人公は怪人に変身する
* 主人公と入管ヒーローとの戦い
* 主人公勝利
* 今後は身を隠しながら生きていかないといけないな、みたいな展望
